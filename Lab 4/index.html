        ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

        Ambos son lenguajes de programación muy importantes y se usan para crear aplicaciones según el tipo de proyecto. Java es un lenguaje para programar orientado a objetos y se usa mucho en sistemas empresariales o aplicaciones grandes y se ejecuta en máquina. JavaScript se ejecuta en navegadores, tiene tipado dinámico y se usa principalmente en el desarrollo web.

        ¿Cómo se declara una variable con alcance local dentro de una función?
    
        Se declara usando let, const o var dentro de la función. De manera que la variable solo existe y se puede usar dentro de esa función.
    
        
    
        ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    
        Se considera una mala práctica porque se pueden modificar desde cualquier parte del programa, lo que puede causar cambios inesperados y desorganización. También aumenta el riesgo de errores, porque varias funciones pueden usar o cambiar la misma variable y reducen la modularidad del código.
