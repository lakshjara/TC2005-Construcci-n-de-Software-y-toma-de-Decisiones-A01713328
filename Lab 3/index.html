Laboratorio 3: CSS

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?
Se recomienda evitar el uso de !important porque puede dificultar el mantenimiento del código. Es mejor organizar bien los estilos y usar selectores adecuados, solo debería utilizarse como último recurso.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?
Si la imagen es muy brillante, oscura o tiene mucho diseño, puede afectar la legibilidad del texto y si una imagen es muy pesada puede hacer que la página cargue más lento.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

  • % es ideal para diseños responsivos, porque se puede adaptar al tamaño de la pantalla.
  • px es útil cuando se necesita precisión en pantalla y control exacto del tamaño.
  • pt se usa principalmente para impresión, no para diseño web.

    • ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?
    Una versión minimizada del CSS ocupa menos espacio, ya que elimina elementos innecesarios, esto ayuda a reducir el tiempo de carga de la página y mejora el rendimiento general del sitio.
